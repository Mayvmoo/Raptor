## Toekomstige Updates

Voor elke update die je wilt pushen naar GitHub:

```bash
# Voeg alle wijzigingen toe
git add -A

# Maak een commit met een beschrijving
git commit -m "Beschrijving van je wijzigingen"

# Push naar GitHub
git push
```

## Handige Git Commands

```bash
# Bekijk de status van je repository
git status

# Bekijk de commit geschiedenis
git log --oneline

# Bekijk welke remote repositories zijn gekoppeld
git remote -v

# Haal de laatste wijzigingen op van GitHub
git pull
```
